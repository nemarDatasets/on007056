There are 2 files contained in this folder. Details of each are below.

BDF_P300_Stim_RTs-mod.txt
This is the bin descriptor file for the task. It contains 4 bins (rare/frequent (2), correct/incorrect(2)). 
These are stimulus-locked bins for targets left or right of fixation with either correct or incorrect 
responses within 200 to 1000ms from stimulus onset. The bdf file also attaches the reaction time for the 
response in the event to the end of all bins 1-4.

PURSUEmastoidref.txt
This is a file that can be used to create a mastoid reference using channel operations for re-referencing 
offline to the average of the mastoids (T9/T10).
